# 📚 College Slides Chat App

A web application that allows users to upload college slides (PDFs), chat with the document content, and manage chat history and personal profile information.

---

## 🚀 Features

### 🔐 Login
- Login via Email & Password

### ℹ️ About
- Information about the application
- How the app works
- Benefits of using document-based chat

### 🏠 Homepage
- Chat interface
- Upload PDF (College Slides)
- Query uploaded PDF
- AI-generated responses


### 🕘 History
- Store chat history
- View previous chats

### ⚙️ Settings
- View personal information
- Update profile details (CRUD)
- Change password
